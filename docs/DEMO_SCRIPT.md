# Demo Script (internal presentation)

Goal: get a visceral “wow” in < 4 minutes.

## Setup (before meeting)
- `make dev`
- Open http://localhost:3000

## Act 1 — The Control Room (30s)
- “This is a shared cognitive substrate for multi-agent teams.”
- Click **Load demo mission**
- Point at HUD:
  - baton holder
  - energy pool
  - live status

## Act 2 — Start simulation (60s)
- Click **Start simulation**
- Narrate:
  - agents explore in parallel
  - they propose patches
  - baton governs commits
  - invariants reject nonsense

## Act 3 — Causal provenance (90s)
- Go to **Causal Graph**
- Click a decision node
- Show:
  - upstream evidence
  - dependency chain
  - invalidation propagation

## Act 4 — Replay and export (60s)
- Go to **Timeline**
- Scrub to a baton transfer
- Then export a Mission Pack

## Optional: Live agents (if configured)
- Use Claude Code team
- Show real writes appearing in UI

## Close (10s)
- “This is where agent teams go next: from message passing to shared typed state + causality + budget arbitration.”
